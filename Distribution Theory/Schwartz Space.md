We adapt the notations and definitions from [[Function on Euclidean Space]] and [[Differential Calculus on Euclidean Space]].

Given a domain $\Omega\subset\mathbb R^d$, a function $f\in\mathcal C^\infty(\Omega)$ belongs to the **Schwartz class** $\mathcal S$ if it satisfies $$\|f\|_{\alpha,\beta}=\sup |x^\alpha D^\beta f|<\infty,\quad\forall\alpha,\beta$$where $x^\alpha=x_1^{\alpha_1}\cdots x_d^{\alpha_d},D^\beta=\partial_1^{\beta_1}\cdots\partial_d^{\beta_d}$, or equivalently, $$|D^\beta f(x)|\le\frac{C_{\beta,N}}{(1+\|x\|)^N},\quad\forall N,\alpha$$Intuitively, the value of $\varphi$ and all its derivatives tend to zero as $x\to\pm\infty$ faster than the inverted value of every polynomial. Note that $\mathcal D\subset\mathcal S$.
- **Convergence in $\mathcal S$**: a sequence $\set{\varphi_i}$ is said to converge in $\mathcal S$ to a function $\varphi\in\mathcal S$ if $$\lim_{i\to\infty}\max_{x\in\mathbb R}(1+|x|)^n|\varphi_i^{(k)}(x)-\varphi^{(k)}(x)|=0,\quad\forall n,k\in\mathbb N$$that is, the convergence is in a (very strong) **uniform** manner even after multiplication by arbitrary polynomials. This is denoted $\varphi_i\xrightarrow{\mathcal S}\varphi$.
- Obviously if $\varphi\in\mathcal S$ then $\varphi'\in\mathcal S$. If $\int_\mathbb R\varphi=0$ then we also have $\int_{-\infty}^x\varphi\in\mathcal S$.
- **Convolution with $L_c^p$**: given $f\in L_c^p$ and $\varphi\in\mathcal S$, by DCT we can perform differentiation arbitrarily many times, hence $f*\varphi\in C^\infty$. For decay condition we have that $$|D^\beta(f*\varphi)(t)|=\left|\int_Kf(t)D^\beta g(x-t)dt\right|\le\int_K|f|\frac{C_{N,\beta}}{(1+|x-t|)^N}dt\le\frac{C'}{(1+|x|)^N}$$This completes the proof.
	- **Counterexample**: taking $f(x)=(1+\|x\|)^{-N}$ with $pN>n$ we have that $f\in L^p(\mathbb R^n)$ while it can be estimated that $|f*\varphi|\ge(2+\|x\|)^{-N}$ for some bump function $\varphi$.
- **Tempered function**: a function $\chi$ is tempered or moderately increasing if there exists $C,m$ such that $|\chi(x)|\le C(1+|x|)^m$ for all $x$. 
	- **Multiplicator**: If $\chi\in\mathcal E$ is tempered and $\varphi\in\mathcal S$, then $\chi\varphi\in\mathcal S$. Such $\chi$ is called a multiplicator on the space $\mathcal S$. Set of multiplicators is usually denoted $\mathcal M$.
