An **arithmetic (number-theoretic) function** is any function $f:\mathbb Z_{\ge1}\to\mathbb C$. Hardy & Wright include in their definition the requirement that an arithmetical function "expresses some arithmetical property of $n$".
- **Additivity/Multiplicativity**: an arithmetic function $f$ is **additive/multiplicative** if $$f(m+n)=f(m)+f(n)\quad\text{or}\quad f(mn)=f(m)f(n),\quad\forall (m,n)=1$$It's called **completely additive/multiplicative** if the above holds for any $m,n\in\mathbb Z_{\ge1}$.
- **Prime power decomposition**: for $n\in\mathbb N,p\in\mathbb P$ define the **$p$-adic valuation function** $$\nu_p(n)=\begin{cases}\max\set{t\in\mathbb N:p^k\mid n},&n\neq0\\\infty,&n=0\end{cases}$$Given $n=p_1^{\alpha_1}\cdots p_k^{\alpha_k}$ define the **prime omega functions** $$\omega(n)=k,\quad\Omega(n)=\sum_{i=1}^k\alpha_k$$
## Euler's totient function
The Euler's totient function $\varphi(n)$ is defined as $$\varphi(n)=\left|\mathbb Z_n^\times\right|=\#\set{1\le m<n:(m,n)=1}$$that is, the [[Cardinality]] of the reduced residue system ([[Modular Arithmetic]]).
- **Closed form expression**: we have the expression $$\varphi(n)=n\prod_{p\mid n,p\in\mathbb P}\left(1-\frac{1}{p}\right)$$we can prove this by verifying its **multiplicity**, reducing it to prime power, which is easy to calculate.
- **Euler's theorem**: for any $(a,n)=1$ we have $$a^{\varphi(n)}\equiv1\pmod{n}$$**Proof**: since $\mathbb Z_n^\times$ forms a group under multiplication, we have $$a\mathbb Z_n^\times=\mathbb Z_n^\times\quad\Longrightarrow\quad\prod_{x\in\mathbb Z_n^\times}x\equiv\prod_{x\in\mathbb Z_n^\times}ax\equiv a^{\varphi(n)}\prod_{x\in\mathbb Z_n^\times}x\pmod n$$by dividing the product on both sides we obtain the desired result.
## Möbius function
The Möbius function is defined by $$\mu(n)=\begin{cases}1,&\text{if }n=1\\(-1)^{\omega(n)},&\text{if }\nu_p(n)=1,\forall p\mid n,p\in\mathbb P\\0,&\text{if }\nu_p(n)>1\text{ for some }p\mid n,p\in\mathbb P\end{cases}$$It can also be represented as $\mu(n)=\delta_{\omega(n)}^{\Omega(n)}\lambda(n)$ where $\lambda(n)$ is the Liouville function. It's obvious from definition that $\mu$ is multiplicative. 
- **Möbius inversion formula**: notice that we have the summation $$\sum_{d\mid n}\mu(d)=\delta_n^1,\quad\forall n\in\mathbb Z_{\ge1}$$hence for any pair of arithmetic functions $f,g$ we have the **Möbius inversion formula** $$g(n)=\sum_{d\mid n}f(d),\quad\forall n\in\mathbb Z_{\ge1}\quad\iff\quad f(n)=\sum_{d\mid n}\mu(d)g\left(\frac{n}{d}\right),\quad\forall n\in\mathbb Z_{\ge1}$$**Proof**: the summation formula follows directly from the fundamental theorem of arithmetics ([[Division]]), and the inversion formula follows by convolving $g$ with $\delta_n^1$ ([[Convolution]]), and change the order or summation. 
	- This result is essentially the same as the inversion formula for [[Fourier Series]] and [[FT on L2]]. 
## Floor & ceiling function
For $x\in\mathbb R$ the **floor** and **ceiling functions** are defined by $$\floor{x}=\max\set{n\in\mathbb Z:n\ge x},\quad\ceil{x}=\min\set{n\in\mathbb Z:n\ge x}$$Based on this we also define the **sawtooth function** $$\set{x}=x-\floor x,\quad\forall x\in\mathbb R$$many properties of these functions follows easily from the definitions.
- **$p$-adic valuation of factorial**: by a simple change of order of summation we have $$\nu_p(n!)=\sum_{j=1}^\infty\floor{\frac{n}{p^j}},\quad\forall p\in\mathbb P$$this is useful when dealing with division problems involving factorials.
