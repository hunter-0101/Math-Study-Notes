Given open $U\subset\mathbb C^n$, a function $f:U\to\mathbb C$ is **holomorphic** if it's locally bounded and holomorphic ([[Holomorphic Function]]) in each variable separately.
- **Cauchy-Riemann equation**: define the Wirtinger operators as in [[Complex Function]] by $$\frac{\partial}{\partial z_k}=\frac{1}{2}\left(\frac{\partial}{\partial x_k}-i\frac{\partial}{\partial y_k}\right),\quad \frac{\partial}{\partial\overline z_k}=\frac{1}{2}\left(\frac{\partial}{\partial x_k}+i\frac{\partial}{\partial y_k}\right)$$then $f$ is holomorphic iff it satisfies the Cauchy-Riemann equation $$\frac{\partial f}{\partial\overline z_k}=0,\quad\forall 1\le k\le n$$
