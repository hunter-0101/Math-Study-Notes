The main idea is that, since given $\mathcal T\subset\mathcal B(V,W)$ UBP ([[Banach Space]]) establish $$\sup_{T\in \mathcal T}\|Tv\|_W<\infty,\forall v\in V\quad\Longrightarrow\quad\sup_{T\in \mathcal T}\|T\|_{\mathcal B(V,W)}<\infty$$by taking the contrapositive argument we know that $$\sup_{T\in\mathcal T}\|T\|_{\mathcal B(V,W)}=\infty\quad\Longrightarrow\quad\exists v\in V\ \st\ \sup_{T\in\mathcal T}\|Tv\|_W=\infty$$and this is quite useful for establishing existence arguments.
- **Constructive discussion**: since in the alternative proof of UBP ([[Banach Space]]) we're  first taking a sequence $T_n\in\mathcal T$ with $\|T_n\|\ge 4^n$, and an associated $v_n\in V$ s.t. $$\|v_n-v_{n-1}\|\le3^{-n}\quad\text{while}\quad \|T_nv_n\|\ge\frac{2}{3}3^{-n}\|T_n\|$$and take $v=\lim v_n$ to contradict assumption, obviously this can be used for constructive proof of related statements.
## Examples
- **Difference between Riemann sum**: there exists $f\in C([0,1])$ s.t. the limit does not exists: $$\lim_{n\to\infty}\left(\sum_{k=1}^{n+1}f\left(\frac{k}{n+1}\right)-\sum_{k=1}^nf\left(\frac{k}{n}\right)\right)$$**Proof**: we defined a linear operator $$T_n:C([0,1])\to\mathbb R,\quad T_nf=\sum_{k=1}^{n+1}f\left(\frac{k}{n+1}\right)-\sum_{k=1}^nf\left(\frac{k}{n}\right)$$and equip $C([0,1])$ with the uniform norm, which makes it complete, then obviously $T_n$ is continuous, hence bounded ([[Operator]]). Notice that we can construct $f_n\in C([0,1])$ with $$f_n\left(\frac{k}{n+1}\right)=1,\quad\forall 1\le k\le n+1\quad\text{while}\quad f_n\left(\frac{k}{n}\right)=0,\quad\forall 1\le k\le n$$and further satisfies $\|f_n\|_\infty=1$, and for such sequence we have $$T_nf_n=n+1\to\infty,\quad n\to\infty$$hence by UBP there exists $f$ with $\sup_nT_nf=\infty$, i.e., the limit above does not exists.
	- **Constructive proof**: firstly if we're not assuming $f\in C([0,1])$ then obviously we can let $$f=\chi_{E}\quad\text{where}\quad E=\bigcup_{p\in\mathbb P}\set{k/n:1\le k\le n}\quad\Longrightarrow\quad T_nf=-n,\quad\forall n=p\in\mathbb P$$to start from here to construct an example in $C([0,1])$, it suffices to use linearity, and adjust the uniform norm so that the convergence is uniform. In particular, we pick $$\mathcal P\subset\mathbb P\quad\st\quad\sum_{p\in\mathcal P}\frac{1}{p}<\infty$$a subset of primes, and define the function by $$f(x)=\sum_{p\in\mathcal P}\frac{\cos(2\pi px)}{p}$$then the convergence is uniform, hence $f\in C([0,1])$, and we can easily verify that $f$ has the desired property.
- **[[Fourier Series]] of continuous function**: there exists $f\in C([0,2\pi])$ whose FS diverges at $0$. The proof via UBP is straightforward.
- **Functional defined by integral**: given $f$ measurable and bounded almost everywhere over a measure space $(X,\mu)$ that is $\sigma$-finite ([[Measure]]) with $\frac{1}{p}+\frac{1}{q}=1$, we have the implication $$Tg=\int_Xfg<\infty,\quad\forall g\in L^q(X)\quad\Longrightarrow\quad f\in L^p(X),T\in(L^q(X))^*,\|T\|=\|f\|_p$$**Proof**: take $E_n\nearrow X,\mu(E_n)<\infty$, and denote the truncated and regularized versions as $$f_n=f\chi_{E_n},\quad g_n=\frac{f_n}{\|f_n\|_q}$$then $\|g_n\|_1=1$. Define the operators $T_n\in(L^q)^*$ by $$T_ng=\int_Xf_ng\quad\Longrightarrow\quad\|T_n\|=\|f_n\|_p$$where the implication follows from HÃ¶lder's inequality ([[Lp Space]]). Now by assumption, $$\sup_{n\in\mathbb N}T_ng=\sup_{n\in\mathbb N}\int_Xf_ng\le\int_Xfg<\infty,\quad\forall g\in L^q$$hence by UBP we have $$\|f\|_p=\sup_{n\in\mathbb N}\|T_n\|<\infty\quad\Longrightarrow\quad f\in L^p$$further, the result in [[Lp Space]] completes the proof.
