Given [[Ring]] $R$, a **left $R$-module** is a triple $(M,+,\cdot)$ of a [[Set]] $M$, an addition $+:M\times M\to M$, and a scaler multiplication $\cdot:R\times M\to M$, denoted $rm$, s.t. for any $r,s\in R,m,n\in M$:
1. **Group structure**: $(M,+)$ is an abelian [[Group]].
2. **Distributivity**: $(r+s)m=rm+sm,r(m+n)=rm+rn$.
3. **Compatibility**: $(rs)m=r(sm)$.
4. **Module identity**: $1x=x$.
Symmetrically we can define **right $R$-module**. A module that is simultaneously a left- and right-$R$ module is usually abbreviated as an **$R$-module**. 
- **Relation with other algebraic structures**: a module can be considered as a pre-[[Vector Space]], and can be turned into one by improve the structure of scalers from a ring to a field. If a module possesses a ring structure, then it's further turned into an associative [[Algebra]]:

- **Submodule**: for an $R$-module $M$, an $R$-submodule is a subgroup $N\le M$ closed under the given module operations. The set of all submodules of $M$, denoted $\sub M$, forms a complete modular [[Lattice]] under inclusion $\subseteq$, where the meet is intersection $\cap$ and the join is the sum $+$.
    - **The Modular Law:** If $N, K, L \in \mathcal{L}(M)$ and $K \subseteq N$, then $N \cap (K + L) = K + (N \cap L)$. This is a critical tool for "diagram chasing" and modularity proofs where vector space-like intuition is applied to rings. 
- **Motivation & ring action**: an $R$-module is the representation of a ring $R$ acting on an abelian [[Group]] $M$. More specifically, similar to [[Group Action]] and [[Group Representation]], any ring can be realized as endomorphism on $M$ via [[Ring]] homomorphism $$\rho: R \hto \text{End}_{R}(M)$$Here $\End_R(M)$ is defined below. For instance, if $R$ is a field $k$, $M$ is a $k$-vector space; if $R = \mathbb{Z}$, $M$ is simply an abelian group.
	- This embedding can be seen as the ring version of Cayley's theorem ([[Group Action]]). 
## Concepts
- **Torsion element**: the collection of torsion elements is defined as $$\tor M=\set{m\in M:rm=0\text{ for some }r\in R,r\neq0}$$Here each element in $\tor M$ is called a **torsion element**. 
	- **Submodule structure over integral domain**: for [[Integral Domain]] $R$, we have $$\tor M\le M$$**Proof**: by commutativity of integral domain, we have $$rm=sn=0,r,s\neq0\imply rs(lm+kn)=sl(rm)+rk(sn)=0$$Here we used the fact that $rs\neq0$ in an integral domain, hence $\tor M\le M$. 
- **Direct sum of modules**: the direct sum of left $R$-modules $\set{M_\alpha}_{\alpha\in I}$ is the left $R^I$-module $$M=\bigoplus_{\alpha\in I} M_\alpha=\Set{(m_\alpha)_{\alpha\in I}\in\prod_{\alpha\in I}M_\alpha:m_\alpha\neq0\text{ for finitely many }\alpha\in I}$$with the element-wise addition & scaler multiplication. 
## Specific types of modules
- **Generated module**: the submodule generated by $S \subseteq M$ is defined as $$\braket{S}=RS=\Set{\sum_{i=1}^{<\infty} r_is_i:r_i\in R,s_i\in. S,\forall i}$$Here $S$ is called the **generator**. It's obvious that generated module is a submodule. 
	- **Finitely generated module**: $N=\braket{S}$ is finitely generated if $|S|<\infty$. 
	- **Cyclic module**: $N=\braket{s}$ is said to be cyclic for any $s\in M$. 
- **Submodule induced by ideal**: we can easily verify that $$IM\le M,\quad\forall I\in\Id R$$It's worth noting that it could happen that $RM<M$, say, $_{2\mathbb Z}\mathbb Z$ is a module with $2\mathbb Z\cdot\mathbb Z=2\mathbb Z$. 
- **Simple (irreducible) module**: $M\neq\set{0}$ is simple if $$\sub M=\set{\set{0},M}$$This definition is similar to that of simple [[Group]]. 
	- **Schur's lemma**: if $M, N$ are simple, any $f \in \text{Hom}_R(M, N)$ is either zero or an isomorphism. Consequently, $\text{End}_R(M)$ is a division ring.
- **Bimodule**: given rings $S,R$, an **$(S,R)$-bimodule** is an abelian group $M$ that is both a left $S$-module and a right $R$-module with $s(mr)=(sm)r$. 
	- **Standard module**: for $R$ commutative, a standard $R$-module is an $(R,R)$-bimodule.
- **Torsion submodule**: $N\le M$ is torsion if $$N\subseteq\tor M$$That is, all its elements are torsion. 
	- **Torsion module**: a module $M$ is torsion if $M=\tor M$. 
	- **Torsion-free module**: $M$ is torsion-free $\tor M=\set{0}$. 
## Module homomorphism
A map ([[Relation]]) $\phi: M \to N$ between two $R$-modules $M,N$ is an **$R$-homomorphism** if it's an $R$-linear [[Group Homomorphism]]: $$\phi(rm + sn) = r\phi(m) + s\phi(n),\quad\forall r,s\in R,m,n\in M$$Unlike ring homomorphisms, module homomorphisms are defined relative to a fixed ring action. 
- **Hom-set**: the set of all homomorphisms between modules is denoted $$\hom_R(M,N)=\set{\phi:M\to N}$$This is obviously an abelian [[Group]]. 
	- **Module structure over commutative ring**: if $R$ is **commutative**, set $$(rf)(m) = r(f(m))$$Then $\text{Hom}_R(M, N)$ inherits an $R$-module structure.
	- **Endomorphism Ring**: the set of endomorphisms, denoted $$\End_R(M) = \hom_R(M, M)$$is a [[Ring]] under addition and function composition. This ring acts on $M$ from the right, making $M$ an $(R, \text{End}_R(M))$-bimodule.
- **Kernel & image**: by properties of [[Group]] and [[Ring]] we have $$\ker\phi\le M,\quad\im\phi\le N$$Therefore homomorphisms are quite commonly used for constructing submodules. 
- **Isomorphism Theorems:**
    1. $M/\ker\phi \cong \im\phi$
    2. $(S + T)/T \cong S/(S \cap T)$
    3. $(M/N)/(K/N) \cong M/K$ for $N \subseteq K \subseteq M$.
## Other info
- **Extension of scalers**: for a ring extension $R\subset S$ and a left $R$-module $N$ we take $M=S$ and define an **extension of scaler** as the [[Tensor]] product $S\otimes_RN$. This structure naturally becomes a left $S$-module under the action of left multiplication.
	- Extension of scalers for free $R$-modules $N$ to the ring $S$ is $$S\otimes_RR^n\cong S^n$$This is shown by considering the free module as direct sum.
- **Quotient module**: for $R$-module $M$ and submodule $N$, the (additive, abelian) quotient group $M/N$ can be made into an $R$-module by defining an action of elements of $R$ by $$r(x+N)=(rx)+N,\quad\forall r\in R,x+N\in M/N$$and the natural projection $\pi:M\to M/N$ by $\pi(x)=x+N$ is an $R$-module homomorphism with kernel $N$. Such $M/N$ is defined as the **quotient module**.
- **Chinese remainder theorem for module**: for given a collection of pairwise coprime [[Ideal]] $\set{I_k}_{k=1}^n\subseteq\Id R$, we have the module isomorphism $$M\left/\Braket{\bigcap_{k=1}^nI_k}\right.\cong\prod_{k=1}^n M/\braket{I_k}$$The proof is essentially the same as that in [[Ideal]].
