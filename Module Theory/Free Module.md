Given a [[Ring]] $R$, a left $R$-[[Module]] $M$ is **free** if it admits a **basis** $B=\set{b_\alpha}_{\alpha\in I}$, i.e., an collection of $R$-linearly independent generators of $M$: $$M=\span_R(b_\alpha:\alpha\in I)$$Equivalently, $F \cong \bigoplus_{\alpha \in I} R$ as $R$-modules for some index set $I$.
- **Universal Property:** A module $F$ is free on $\mathcal{B}$ if and only if for every $R$-module $M$ and every set map $f: \mathcal{B} \to M$, there exists a unique $R$-linear homomorphism $\varphi: F \to M$ such that $\varphi|_\mathcal{B} = f$. This makes $F$ the "most general" module generated by $\mathcal{B}$, having no relations other than those forced by the module axioms.
- **Intermediate algebraic status**: a free module sits between a general $R$-module and a [[Vector Space]].
    - **vs. vector space**: while every vector space has a basis, not every module is free (e.g., $\mathbb{Z}/n\mathbb{Z}$ as a $\mathbb{Z}$-module has torsion and no basis).
    - **vs. general module**: free modules lack "pathological" relations. In a general module $M$, $rx=0$ does not imply $r=0$ or $x=0$. In a free module, linear independence ensures that $\sum r_i b_i = 0 \implies r_i = 0$ for all $i$.
- **Projectivity:** Every free module is **projective**. This implies that any surjective homomorphism $p: M \to F$ splits, meaning $M \cong \ker(p) \oplus F$. This is a primary tool in homological algebra and the study of extension groups. 
## Properties
- **Invariant basis number (IBN) property**: for a commutative ring $R$, we have $$R^m\cong R^n\text{ as }R\text{-modules}\qiffq m=n$$**Proof**: for $(\Rightarrow)$, let $\mathfrak{m}$ be a maximal [[Ideal]] of $R$. If $R^m \cong R^n$, then we can verify easily that $$(R/\mathfrak m)^m\cong R^m / \mathfrak{m} R^m \cong R^n / \mathfrak{m} R^n\cong(R/\mathfrak m)^n$$Since $R/\mathfrak m$ is a field, by [[Vector Space]] we have $m=n$. $(\Leftarrow)$ is trivial. 
- **Rank**: IBN property allows us to define the rank of a free module as ([[Cardinality]]) $$\rank(M) = \#B$$For finitely generated free modules, $\text{rank}(F) \in \mathbb{N}$.
	- **Canonical isomorphism**: given IBN property, we have $$M\cong R^{\rank M}$$This is similar to the canonical isomorphism of [[Vector Space]]. 
- **PID submodule theorem**: if $R$ is a [[Principal Ideal Domain]], every submodule $N$ of a free module $M$ is also free, with rank inequality $$\rank N\le\rank M$$**Proof**: 
	- For general $R$, a submodule of a free module is not necessarily free. 

- **Relation to dual module**: if $F$ is a finitely generated free module of rank $n$, then its dual $F^* = \text{Hom}_R(F, R)$ is also a free module of rank $n$. The evaluation map $F \to F^{**}$ is an isomorphism, a property generally shared only by reflexive modules.
## Examples
- **[[Polynomial Ring]] $R[x]$:**
    - **Analysis:** $R[x]$ is a free $R$-module with an infinite basis $\mathcal{B} = \{1, x, x^2, \dots, x^n, \dots\}$.
    - **Insight:** This illustrates that free modules can have infinite rank. It also serves as the free $R$-algebra, where the basis elements carry additional multiplicative structure.
- **Gaussian Integers $\mathbb{Z}[i]$ over $\mathbb{Z}$:**
    - **Analysis:** $\mathbb{Z}[i] = \{a + bi \mid a, b \in \mathbb{Z}\}$ is a free $\mathbb{Z}$-module of rank 2 with basis $\{1, i\}$.
    - **Insight (Angle A - Geometric/Structural):** From a structural perspective, $\mathbb{Z}[i] \cong \mathbb{Z}^2$. This shows how algebraic integers in number fields can be treated as free modules over $\mathbb{Z}$, allowing for the use of "lattice-like" arguments in algebraic number theory.
- **The Ideal $I = (2, x)$ in $\mathbb{Z}[x]$:**
    - **Analysis:** Consider $I$ as a $\mathbb{Z}[x]$-module. $I$ is **not** a free $\mathbb{Z}[x]$-module.
    - **Insight (Angle B - Contrast via Relations):** While $I$ is generated by $\{2, x\}$, these generators are not linearly independent: $x(2) + (-2)(x) = 0$, but $x, -2 \neq 0$ in $\mathbb{Z}[x]$. This demonstrates that even in "nice" rings (UFDs or Noetherian rings), non-principal ideals typically fail to be free modules, highlighting the rigidity of the freeness condition.
