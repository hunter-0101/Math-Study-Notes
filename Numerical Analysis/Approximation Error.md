Given a true value $v$, for the approximation value $\tilde v$ we define the following concepts:
- **Absolute error**: $\varepsilon$ for $|v-\tilde v|\le\varepsilon$.
- **Relative error**: $\eta$ for $|v-\tilde v|\le\eta\cdot v$.
- **Percentage error**: $\delta$ for $\delta=\eta\times100\%$.
- **Error bound**: upper limit for one of the above error kind.
- **Significant digits**: given a true value $x$ and an approximating value $\tilde x$, the significant digits include all digits that start from the first non-zero entry of $x$ and ends at the first non-zero entry of $|x-\tilde x|$. E.g., for $x=0.51234,\tilde x=0.51220$ we have $|x-\tilde x|=0.00014$, hence the significant digits are $0.\textbf{000}14$.
## Machine error
For a $B$-based system with precision $P$, lower/upper exponent $L,U$, the family of expressible floating numbers is $$F(B,P,L,U)=\set{\operatorname{fl}(x)=\pm x_1\cdots x_P\times B^l:L\le l\le U}$$Notice that $$F_\max=B^{U+1}(1-B^{-P}),\quad F_\min=B^L$$any number that exceeds these two bounds would cause **overflow/underflow**.
## Condition number
Given a problem $f$ and an approximation algorithm $\tilde f$, the error is $\delta f(x)=f(x)-\tilde f(x)$. We define the **absolute/relative condition number** as $$\lim_{\varepsilon\to 0^+}\sup_{\|\delta x\|\le\varepsilon}\frac{\|\delta f(x)\|}{\|\delta x\|}\qquad\text{and}\qquad\lim_{\varepsilon\to 0^+}\sup_{\|\delta x\|\le\varepsilon}\frac{\|\delta f(x)\|/\|f(x)\|}{\|\delta x\|/\|x\|}$$It measures how much the output changes with a small change in the input, i.e., how sensitive the algorithm is to small errors. A problem with low/high condition number is called **well-/ill-conditioned**, and the output w.r.t a small disturbance in input is stable/unstable.
- **Matrix**: for a system $Ax=b$ where $A$ is non-singular, let $e$ be error in $b$, then $$\kappa(A)=\max_{e,b\neq0}\frac{\|A^{-1}e\|/\|e\|}{\|A^{-1}b\|/\|b\|}=\max_{e\neq0}\frac{\|A^{-1}e\|}{\|e\|}\max_{b\neq0}\frac{\|b\|}{\|A^{-1}b\|}=\|A^{-1}\|\cdot\|A\|$$where $\|\cdot\|$ is any operator norm ($\kappa_p$ for $\|\cdot\|_p$). 
	- **$2$-condition number**: by [[Matrix]] we have that $$\kappa_2(A)=\sqrt\frac{\lambda_\max(A^*A)}{\lambda_\min(A^*A)}=\frac{\sigma_\max(A)}{\sigma_\min(A)}$$where $\sigma$ denotes the singular value, since $$\|A^{-1}\|=\lambda_\max((A^{-1})^*A^{-1})=\lambda_\min^{-1}(AA^*)=\lambda_\min^{-1}(A^*A)$$by the fact that $\lambda\in\sigma(AA^*)\iff\lambda\in\sigma(A^*A)$ ([[Singular Value Decomposition]]).
- **Non-linear**: give any function $f:\mathbb R^{d_1}\to\mathbb R^{d_2}$, we could define $$\kappa(f)=\lim_{\varepsilon\to0^+}\sup_{\|\delta x\|\le\varepsilon}\left(\frac{\|f(x+\delta x)-f(x)\|}{\|f(x)\|}\Big/\frac{\|\delta x\|}{\|x\|}\right)=\frac{\|J(x)\|}{\|f(x)\|/\|x\|}$$where $J(x)$ is the Jacobian.
