A **[[Field]] extension** is an inclusion map $$\iota:K\to L$$which will be denoted $L/K$, with $L$ being the **extension field** and $K$ being the **subfield**. When there is a chain $L/M/K$ we call $M$ the **intermediate field**. 
- **Degree**: by viewing $L$ as a $K$-[[Vector Space]], the degree of $L/K$ is defined as $$[L:K]=\dim_KL$$An extension of finite degree is called a **finite extension**.
	- **Short tower law**: given $K\subset M\subset L$ we have $$[L:K]=[L:M][M:K]$$**Proof**: take $B_{L/M}=\set{l_i},B_{M/K}=\set{m_j}$ the basis s.t. $$L=\span_M(B_{L/M}),\quad M=\span_K(B_{M/K})$$then obviously $B=B_{L/M}B_{M/K}$ forms a $K$-frame for $L$. For linear dependence notice that $$\sum_{i,j}k_{ij}l_jm_j=0\quad\Longrightarrow\quad\sum_j\left(\sum_i k_{ij}l_j\right)m_i=0\quad\Longrightarrow\quad\sum_ik_{ij}l_j=0\quad\Longrightarrow\quad k_{ij}=0,\forall i,j$$hence $B$ is indeed a $K$-basis for $L$, completing the proof.
	- **Tower law**: by induction we can easily show that $$[K_n:K_0]=\prod_{i=1}^n[K_i:K_{i-1}]$$This is quite useful for determining the degree of an extension. 
## Extension by field generation
Since the set of intermediate fields is closed under intersection, given any set $S\subset L$ there exists a **unique** minimal intermediate field $K(S)$ s.t. $$S\subset K(S),\quad K\subset K(S)\subset L$$We call $K(S)$ the **field generated by $S$ over $K$**. $K(S)$ is **finitely generated** if $|S|<\infty$.
- **Alternative characterization**: we have the field of fraction characterization $$K(\alpha_1,\cdots,\alpha_s)=\Frac (K[\alpha_1,\cdots,\alpha_s])$$where $K[\alpha_1,\cdots,\alpha_s]$ is the multivariate [[Polynomial Ring]], since all elements in RHS can be obtained via algebraic calculation, while it's indeed a field.
- **Simple extension**: $L/K$ is simple if $L=K(\alpha)$ for some $\alpha\in L$. We have $$[K(\alpha):K]=\begin{cases}\deg\mu_{\alpha/K},&\text{if }\alpha\text{ is algebraic}\\\infty,&\text{if }\alpha\text{ is transcendental}\end{cases}$$by considering the set $\alpha^n,n\in\mathbb N$. The proof is detailed below. 
	- **Algebraic simple extension**: given $L/K$ and $\alpha\in L$ we have $$K(\alpha)/K\text{ is algebraic/finite}\iff\alpha \text{ is algebraic over }K\iff K[\alpha]=K(\alpha)$$**Proof**: we loop along $\Rightarrow$ to establish the equivalence. If $K(\alpha)/K$ is algebraic we have $\alpha$ algebraic over $K$ by definition; if $\alpha$ is algebraic, consider $\mu_{\alpha/K}$ ([[Polynomial over a Field]]), then for any $f\in K[x]$ we first use the Euclidean division ([[Polynomial Ring]]) to reduce to the case $\deg f<\deg\mu_{\alpha/K}$, and then use BÃ©zout's identity ([[Polynomial]]) to get $$pf+q\mu_{\alpha/K}=1\quad\Longrightarrow\quad p(\alpha)f(\alpha)=1$$hence $(f(\alpha))^{-1}\in K[\alpha]$, which implies $K[\alpha]=K(\alpha)$; Finally to show $\dim_KK(\alpha)<\infty$ just notice that $\set{1,\alpha,\cdots,\alpha^{d-1}},d=\deg\mu_{\alpha/K}$ forms a $K$-basis for $K(\alpha)=K[\alpha]$.  
	- **Induction trick**: by definition given $L/K$ and $\alpha,\beta\in L$ we have $$K(\alpha,\beta)=K(\alpha)(\beta)$$Hence induction can be used when proving results for finitely generated field. This is immediately applicable to the next result for $\Leftarrow$.
- **Criterion for finite extension**: $L/K$ is finite iff it's algebraic with $L=K(\alpha_1,\cdots,\alpha_s)$. 
  **Proof**: given $L/K$ finite, if $\alpha\in L$ is not algebraic over $K$ then $\set{1,\alpha,\alpha^2,\cdots}$ forms an infinite basis, which is a contradiction, and for the expression it suffices to take $\alpha_i$ the $K$-basis for $L$; conversely it suffices to use induction on $s$, based on the result on simple extension above.
	- Notice that finite $\Rightarrow$ algebraic, but the converse is not true in general.
## Algebraic & transcendental extension
An extension $L/K$ is **algebraic** if all elements in $L$ are algebraic over $K$ ([[Field]]).
- **Field of algebraic elements**: give $L/K$, there exists an intermediate field constructed as $$M=\set{\ell\in L:\ell\text{ is algebraic over }K}$$**Proof**: obviously $K\subset M\subset L$. Notice that for any $\alpha,\beta\in M$ we have $$[K(\alpha,\beta):K]=[K(\alpha,\beta):K(\alpha)][K(\alpha):K]<\infty$$by the simple algebraic extension result below, hence $K(\alpha,\beta)/K$ is algebraic, which means $\alpha\pm\beta,\alpha\beta,\alpha\beta^{-1}\in M$, hence $M$ is a field.
- **Algebraic over algebraic is algebraic**: $L/M,M/K$ both algebraic implies $L/K$ algebraic.
  **Proof**: for any $\alpha\in L$, since $L/M$ is algebraic there exist $b_i\in M$ s.t. $\sum_{i=0}^nb_i\alpha^i=0$, hence $$\begin{align}[K(\alpha):K]&\le[K(b_0,\cdots,b_n,\alpha):K]\\&=[K(b_0,\cdots,b_n,\alpha):K(b_0,\cdots,b_n)][K(b_0,\cdots,b_n):K]\\&<\infty\end{align}$$Where we used the tower law above and the finite extension criterion below, hence $\alpha$ is algebraic over $K$. By arbitrariness $L/K$ is algebraic.
- **Examples of infinite algebraic extension** 
	- **Roots of unity**: consider $$\mathbb Q(U)/\mathbb Q\quad\text{where}\quad U=\set{z\in\mathbb C:z^p=1,p\in\mathbb P}$$This is infinite because any set of $p$-th roots of unity contributes $p-1$ degrees, and is algebraic due to some simple identities.
	- **Square root of primes**: consider $$\mathbb Q(\sqrt{\mathbb P})/\mathbb Q\quad\text{where}\quad\sqrt{\mathbb P}=\set{\sqrt p:p\in\mathbb P}$$This is infinite because elements in $\sqrt{\mathbb P}$ are $\mathbb Q$-linearly independent (we skip the proof).
