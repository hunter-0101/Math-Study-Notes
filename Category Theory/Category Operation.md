## Object operations
- **Coproduct**: an object is called the coproduct of $X_1,X_2\in\ob(\mathcal C)$, denoted $X_1\coprod X_2$, if there exists morphisms $i_1:X_1\to X_1\coprod X_2,i_2:X_2\to X_1\coprod X_2$ with the [[Universal Property]] that for any $Y\in\ob(\mathcal C),f_1:X_1\to Y,f_2:X_2\to Y$ there exists unique $f:X_1\coprod X_2\to Y$ s.t.
